The Dutch Wisdom Bot is inspired by our good old friend Inspirobot. 
One of the things that makes Inspirobot a bit tricky sometimes is that it includes NSFW words, questionable images, and sometimes very negative words - it can be a thrill, but it's not for everyone.
The Dutch Wisdom Bot provides toggles for including NSFW and negative content, so that you can choose what you're in the mood for that day.

You need Pyside6 and Pillow to be installed on your machine to make it work.
